1. Application Foundation (how Laravel actually works)
2. System Design (architecture & scaling)
3. Platform Engineering (infra & tooling)
4. Advanced Intelligence Systems (AI + analytics)

Each domain will later be expanded into:
- Subdomains
- Knowledge Units
- Rules
- Skills
- Decision Trees
- Anti-Patterns
