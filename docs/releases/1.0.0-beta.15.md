# LaraSkills 1.0.0-beta.15

## Summary

This release prepares the complete Laravel ECC to LaraSkills migration while
preserving a temporary compatibility bridge for existing installations.

## Rebrand

- Product name: LaraSkills
- Repository target: `elmochilyas/laraskills`
- Version: `1.0.0-beta.15`
- Primary log prefixes and user-facing output now use LaraSkills.

## npm Package Migration

The package name changes from `laravel-ecc` to `laraskills`. Repository,
homepage, issue tracker, package files, postinstall guidance, and lockfile
metadata now point at the LaraSkills identity.

## CLI and MCP Migration

Primary binaries:

```text
laraskills
laraskills-mcp
```

The CLI adds `--laraskills-root`, and the MCP server metadata name is
`laraskills`. Preferred environment variables are `LARASKILLS_ROOT` and
`LARASKILLS_CONFIG_DIR`.

## Compatibility Bridge

The package temporarily retains:

```text
laravel-ecc
laravel-ecc-mcp
--ecc-root
ECC_ROOT
LARAVEL_ECC_CONFIG_DIR
laravel-ecc/config.json
eccRoot config field
```

Preferred inputs take precedence, and doctor identifies active legacy
fallbacks.

## Phase 16 Cross-Platform CI

The release retains the Phase 16 Windows, Ubuntu, and macOS matrix across Node
18, 20, and 22, plus packed-install, MCP smoke, and reproducibility jobs.

## Clean-Clone Reproducibility

Clean-clone verification continues to rebuild intelligence twice, compare
hashes, enforce PowerShell availability in strict CI, run tests, and run the
retrieval benchmark suite.

## Dependency Injection Performance Improvements

The Phase 16 baseline includes the previously merged dependency-edge injection
performance and cross-platform path handling improvements. This rebrand does
not change graph semantics.

## Known Limitations

- The GitHub repository rename is not performed by this local preparation.
- The npm package is not published by this local preparation.
- The old npm package is not deprecated until the new package is verified.
- Compatibility aliases are temporary and will be removed in a later release.

## Upgrade Notes

Install `laraskills`, update CLI and MCP names, migrate root environment
variables, run `laraskills setup`, then verify with `laraskills doctor` and
`laraskills validate`. See
[`../migrations/laravel-ecc-to-laraskills.md`](../migrations/laravel-ecc-to-laraskills.md).
