## Retrieval Budgeting and Convergence Guidance

- Standard bundle first, validate once, then targeted searches only
- Normal tasks: 2â€“10 MCP calls
- Complex tasks: 10â€“20 MCP calls
- More than 20 calls triggers convergence review

## Multi-Tenant Isolation Hardening

- Query-level isolation (`tenant_id` in every WHERE clause) is the primary isolation boundary
- Global scopes provide defense in depth but must not be the only mechanism
- Tenant-aware validation: middleware resolves tenant from secure sources (subdomain/header/auth), never from user input
- Correct relationship scoping: all relationships must cascade tenant context
- Cross-tenant leakage tests required for every multi-tenant feature

## Package Distribution Strategy

The npm package intentionally excludes the full knowledge and intelligence layers. Retrieval requires access to a full Laravel ECC checkout. Run `laravel-ecc setup` to configure the checkout path.
