## Known Beta Limitations

- Full knowledge layer is intentionally excluded from npm.
- ECC_ROOT must point to a full Laravel ECC checkout or compatible knowledge bundle.
- Remote MCP hosting is not implemented.
- Semantic embedding fallback is not implemented.
- Broader knowledge-quality backlog remains deferred.
- This is a beta release intended for Phase 11.3 real-project testing.
