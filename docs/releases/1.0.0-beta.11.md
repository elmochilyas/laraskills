# Laravel ECC 1.0.0-beta.11

## Summary

Metadata-only follow-up release correcting the npm repository, homepage, and issue-tracker links.

## Changes

- Repository metadata now points to `https://github.com/elmochilyas/laravel-ecc`
- Homepage now points to the maintained project README
- Bug reports now point to the maintained project issue tracker
- No CLI, MCP, retrieval, knowledge, or intelligence behavior changed

## Validation

This release inherits the verified `1.0.0-beta.10` implementation:

- 150 / 150 tests passing
- 71 / 71 benchmarks passing
- 2,321 knowledge units
- 429 dependency edges
- 3,513 relationship edges
- Intelligence layer VALID

## Package Distribution Strategy

The npm package remains intentionally lightweight.
The full knowledge and intelligence layers are not bundled.
Set `ECC_ROOT` when the full Laravel ECC checkout cannot be resolved automatically.

## Upgrade Notes

Install the current beta with:

```bash
npm install laravel-ecc@beta
```

No intentional CLI or MCP API behavior changes were introduced in this metadata-only release.
