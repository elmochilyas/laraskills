## Important Distribution Note

The npm package is intentionally lightweight.

The full knowledge and intelligence layers are not bundled into npm.

Users must set:

ECC_ROOT=<path-to-full-laravel-ecc-repository>

or point ECC_ROOT to a compatible knowledge bundle.
