LaraSkills — Laravel 13 knowledge for AI coding agents — is now in public beta.

LaraSkills gives AI coding tools two layers of Laravel-specific intelligence:

1. An operating layer with 12 skills, 41 rules, and 12 agent definitions you install in any Laravel project with one command. It trains your AI tool on Laravel 13 conventions, security patterns, and architecture standards.

2. A knowledge intelligence layer with 2,321 curated knowledge units across 21 engineering domains — Eloquent, APIs, auth, testing, databases, security, queues, real-time, and more. Accessible through a deterministic CLI and an MCP server for real-time retrieval during AI sessions.

Laravel 13, PHP 8.3, Pest 4. Works with OpenCode, Claude Code, Cursor, Gemini CLI, Codex CLI, GitHub Copilot, VS Code with MCP, and others.

```bash
npm install --save-dev laraskills
npx laraskills install --profile core
```

The package is validated against fresh and existing Laravel 13 projects with 202 passing tests and 72 retrieval benchmarks at 100% accuracy.

Looking for developers to test it in real projects and share feedback. If you build Laravel apps with AI assistance, give it a try.
