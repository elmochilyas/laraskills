# Decomposition: vector embeddings concept

## Topic Overview

Vector embeddings convert text (or images, audio, etc.) into numerical vector representations that capture semantic meaning. Similar concepts produce similar vectors, enabling semantic search. Embeddings are generated by neural network models (transformers) and stored in vector databases or indexes for similarity search.
