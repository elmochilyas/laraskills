# Skills: Vault Secrets Management

## Skill: doppler-laravel-integration
**Purpose:** Integrate Doppler vault with Laravel
**Trigger:** When implementing centralized secrets for Laravel
**Workflow:**
1. Create Doppler account and project
2. Configure environments in Doppler
3. Install Doppler Laravel SDK
4. Map Doppler secrets to Laravel config
5. Set up CLI fallback for development
6. Configure CI/CD integration
7. Test secret resolution in all environments
**Output:** Doppler-integrated Laravel application

## Skill: aws-secrets-manager-integration
**Purpose:** Integrate AWS Secrets Manager with Laravel
**Trigger:** When using AWS infrastructure for Laravel
**Workflow:**
1. Create secrets in AWS Secrets Manager
2. Configure IAM role for secret access
3. Implement Laravel config provider for Secrets Manager
4. Set up automatic rotation schedule
5. Cache secrets in Laravel cache
6. Implement fallback for vault outage
**Output:** AWS Secrets Manager integration with Laravel
