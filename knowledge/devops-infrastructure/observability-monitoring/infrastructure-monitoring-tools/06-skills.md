# Skills: Infrastructure Monitoring Tools

## Skill: observability-stack-setup
**Purpose:** Set up observability stack for Laravel application
**Trigger:** When establishing production monitoring
**Workflow:**
1. Choose monitoring tools (Nightwatch, Pulse, Sentry, Datadog)
2. Configure structured logging
3. Implement health check endpoint
4. Set up Pulse dashboard
5. Integrate error tracking (Sentry/Flare)
6. Configure alert thresholds
7. Set up notification channels (Slack, email, PagerDuty)
8. Establish on-call rotation
**Output:** Production observability stack with alerting
