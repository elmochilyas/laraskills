# Skills: Observability & Monitoring

## Skill: slo-driven-alerting
**Purpose:** Set up SLO-driven alerting for Laravel
**Trigger:** When establishing production monitoring with measurable targets
**Workflow:**
1. Define SLOs (uptime 99.9%, error rate < 1%, p95 latency < 500ms)
2. Configure burn-rate alerting
3. Set up multi-window, multi-burn-rate alerts
4. Define SLIs for each SLO
5. Create error budget dashboard
6. Integrate with incident response
**Output:** SLO-driven alerting with error budget tracking
