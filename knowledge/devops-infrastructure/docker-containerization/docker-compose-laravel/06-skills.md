# Skills: Docker Compose Laravel

## Skill: docker-compose-laravel-env
**Purpose:** Create Docker Compose environment for Laravel development
**Trigger:** When setting up local development with Docker
**Workflow:**
1. Define services (PHP-FPM, Nginx, MySQL, Redis)
2. Configure Dockerfile for PHP with extensions
3. Set up Nginx configuration for Laravel
4. Use named volumes for database persistence
5. Add health checks for dependency ordering
6. Configure environment variables per service
7. Create Makefile or sail script for common commands
**Output:** Docker Compose environment for Laravel development
