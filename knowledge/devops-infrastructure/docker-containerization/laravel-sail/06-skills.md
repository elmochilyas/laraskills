# Skills: Laravel Sail

## Skill: sail-environment-setup
**Purpose:** Set up and configure Laravel Sail for team development
**Trigger:** When initializing a new Laravel project with Docker development
**Workflow:**
1. Install Sail via Composer
2. Configure services (database, cache, mail, search)
3. Set PHP version and extensions
4. Publish and customize Sail configuration
5. Create `.env` with Sail-specific settings
6. Test environment with `sail up`
7. Document customizations for team
**Output:** Standardized Docker development environment for team
