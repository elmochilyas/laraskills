# Skills: Rollback Strategies

## Skill: database-rollback-procedure
**Purpose:** Create database rollback procedure for Laravel deployments
**Trigger:** When establishing deployment rollback process
**Workflow:**
1. Ensure all migrations are reversible
2. Test rollback on staging with production-like data volume
3. Implement rollback trigger in CI/CD pipeline
4. Document rollback procedure for incident response
5. Train team on rollback execution
6. Schedule periodic rollback drills
**Output:** Documented, tested database rollback procedure
