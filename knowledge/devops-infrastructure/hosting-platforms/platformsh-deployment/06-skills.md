# Skills: Platform.sh Deployment

## Skill: platformsh-laravel-setup
**Purpose:** Deploy Laravel on Platform.sh
**Trigger:** When setting up enterprise Laravel hosting on Platform.sh
**Workflow:**
1. Create Platform.sh account and project
2. Create `.platform.app.yaml` with Laravel configuration
3. Configure `.platform/services.yaml` for database/Redis
4. Set up `.platform/routes.yaml` for routing
5. Push Git repository to trigger initial deployment
6. Configure development environments per branch
7. Set up environment variables
8. Enable CDN and SSL
**Output:** Laravel deployed on Platform.sh
