# Skills: Forge

## Skill: forge-platform-setup
**Purpose:** Set up Forge as hosting platform
**Trigger:** When using Forge for Laravel hosting
**Workflow:**
1. Create Forge account and connect cloud provider
2. Create Forge recipes for provisioning
3. Set up team access
4. Provision production server
5. Configure Nightwatch monitoring
6. Set up deployment via Forge API
7. Configure backup schedule
**Output:** Forge-managed Laravel hosting environment
