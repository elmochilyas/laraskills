# Skills: DigitalOcean App Platform

## Skill: do-app-platform-deploy
**Purpose:** Deploy Laravel on DigitalOcean App Platform
**Trigger:** When setting up Laravel on App Platform
**Workflow:**
1. Create App Platform resource in DigitalOcean
2. Connect GitHub/GitLab repository
3. Configure Dockerfile or buildpacks
4. Set environment variables
5. Provision managed database
6. Configure health check endpoint
7. Enable auto-deploy on Git push
8. Set up CDN for static assets
**Output:** Laravel deployed on DigitalOcean App Platform
