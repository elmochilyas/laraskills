# Skills: Envoyer

## Skill: envoyer-project-admin
**Purpose:** Configure and manage Envoyer project
**Trigger:** When setting up Envoyer as deployment platform
**Workflow:**
1. Create Envoyer account
2. Create project and connect Git repository
3. Add servers with SSH keys
4. Configure deployment hooks
5. Set up team member access
6. Integrate with CI/CD pipeline via API
7. Configure Slack notifications
8. Test deployment pipeline
**Output:** Envoyer project with CI/CD integration
