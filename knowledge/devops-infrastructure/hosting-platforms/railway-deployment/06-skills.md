# Skills: Railway Deployment

## Skill: railway-laravel-deploy
**Purpose:** Deploy Laravel on Railway
**Trigger:** When setting up Laravel on Railway
**Workflow:**
1. Create Railway account
2. Connect GitHub repository
3. Railway auto-detects Laravel via Nixpacks
4. Provision PostgreSQL database
5. Configure environment variables
6. Set up health check
7. Configure custom domain
8. Enable auto-deploy on Git push
**Output:** Laravel deployed on Railway
