# Skills: Platform Selection

## Skill: hosting-platform-evaluation
**Purpose:** Evaluate and select hosting platform for Laravel
**Trigger:** When choosing hosting for new or migrating Laravel application
**Workflow:**
1. Identify application requirements (traffic, regions, compliance)
2. Assess team operational capabilities
3. Shortlist platforms matching requirements
4. Calculate total cost for each option
5. Evaluate migration difficulty
6. Run load tests on shortlisted platforms
7. Make recommendation with rationale
**Output:** Hosting platform recommendation with cost analysis
