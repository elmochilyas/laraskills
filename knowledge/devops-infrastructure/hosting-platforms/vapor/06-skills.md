# Skills: Vapor

## Skill: vapor-platform-setup
**Purpose:** Set up Laravel Vapor as hosting platform
**Trigger:** When deploying Laravel to serverless on Vapor
**Workflow:**
1. Install Vapor CLI
2. Create `vapor.yml` with application and environment config
3. Configure databases and cache
4. Set up teams and access control
5. Integrate with CI/CD pipeline
6. Enable config cache for cold start optimization
7. Set up cost monitoring alerts
8. Deploy staging environment for testing
**Output:** Vapor-managed Laravel serverless deployment
