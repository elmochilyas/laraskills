# Skills: Fly.io Deployment

## Skill: flyio-laravel-deploy
**Purpose:** Deploy Laravel on Fly.io
**Trigger:** When setting up Laravel on Fly.io platform
**Workflow:**
1. Install Fly.io CLI and authenticate
2. Create Dockerfile for Laravel production
3. Configure `fly.toml` with app settings
4. Provision managed PostgreSQL database
5. Set environment variables
6. Configure health checks
7. Enable auto-scaling
8. Deploy with `fly deploy`
**Output:** Laravel deployed on Fly.io edge network
