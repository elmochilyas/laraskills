# Skills: Terraform AWS Laravel

## Skill: terraform-aws-infrastructure
**Purpose:** Provision Laravel infrastructure on AWS with Terraform
**Trigger:** When setting up AWS infrastructure for Laravel
**Workflow:**
1. Configure S3 backend with DynamoDB locking
2. Define VPC with public/private subnets
3. Provision RDS MySQL/PostgreSQL
4. Provision ElastiCache Redis
5. Set up ECS Fargate cluster or EKS
6. Create S3 buckets with IAM policies
7. Configure CloudFront distribution
8. Set up IAM roles for deployment
9. Apply consistent tagging
**Output:** Terraform configuration for Laravel on AWS
