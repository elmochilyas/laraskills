# Skills: Terraform Basics

## Skill: terraform-project-setup
**Purpose:** Set up Terraform project for Laravel infrastructure
**Trigger:** When initiating infrastructure provisioning with Terraform
**Workflow:**
1. Install Terraform and configure cloud provider
2. Create directory structure for environments
3. Configure remote state backend
4. Define provider and required version
5. Create reusable modules for common patterns
6. Parameterize with input variables
7. Tag all resources with consistent tags
8. Document outputs for dependent systems
**Output:** Terraform project with environment separation
