# Skills: Pulumi for Laravel

## Skill: pulumi-laravel-infrastructure
**Purpose:** Create Pulumi program for Laravel cloud infrastructure
**Trigger:** When provisioning Laravel infrastructure with Pulumi
**Workflow:**
1. Initialize Pulumi project with TypeScript/Python/Go
2. Configure state backend (S3 or Pulumi Cloud)
3. Create stacks for each environment
4. Define VPC, subnets, and networking
5. Provision RDS database
6. Provision ElastiCache Redis
7. Provision ECS Fargate or EKS cluster
8. Set up S3 for file storage
9. Configure CloudFront CDN
10. Deploy with `pulumi up`
**Output:** Laravel infrastructure provisioned with Pulumi
