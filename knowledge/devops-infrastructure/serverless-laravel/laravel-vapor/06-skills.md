# Skills: Laravel Vapor

## Skill: vapor-deployment-setup
**Purpose:** Set up Laravel deployment on Vapor
**Trigger:** When deploying Laravel to serverless on Vapor
**Workflow:**
1. Install Vapor CLI
2. Create `vapor.yml` with application configuration
3. Define environments and database configurations
4. Configure cache commands in deploy hooks
5. Set up queue workers for job processing
6. Configure CDN for asset delivery
7. Deploy to staging environment
8. Test and promote to production
**Output:** Vapor-deployed Laravel application with auto-scaling
