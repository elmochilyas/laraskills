# Skills: Bref Laravel

## Skill: bref-laravel-setup
**Purpose:** Set up Laravel on AWS Lambda with Bref
**Trigger:** When deploying Laravel as serverless application
**Workflow:**
1. Install Bref via Composer
2. Configure `serverless.yml` with HTTP and CLI functions
3. Set up API Gateway for web routes
4. Configure SQS queue for job processing
5. Implement configuration cache in deployment
6. Set up RDS Proxy for database connections
7. Configure S3 for file uploads
8. Optimize Composer autoloader
**Output:** Bref deployment configuration for Laravel on Lambda
