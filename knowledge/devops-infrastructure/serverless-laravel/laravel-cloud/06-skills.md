# Skills: Laravel Cloud

## Skill: cloud-deployment-setup
**Purpose:** Set up Laravel deployment on Laravel Cloud
**Trigger:** When migrating or setting up new project on Cloud
**Workflow:**
1. Create Cloud account and project
2. Connect Git repository
3. Configure environment variables
4. Set up database and cache services
5. Configure deployment triggers
6. Enable hibernation for development environments
7. Set up custom domain with SSL
8. Monitor initial deployment
**Output:** Laravel Cloud deployment with Git-based workflow
