# Executive Summary

Telescope and Pulse serve complementary roles: Telescope for local/staging request-level debugging, Pulse for production aggregate application metrics. This unit clarifies when to use each, when both are warranted, and when neither replaces proper APM (Sentry/Bugsnag/Datadog).
