# Executive Summary

Spatie laravel-permission is the de facto standard for database-driven roles and permissions in Laravel. It fits for most multi-role applications. It may not fit when roles are embedded in JWT tokens, when authorization is purely relationship-based, or when a ReBAC (relationship-based access control) model is more natural than RBAC.
